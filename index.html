Random Number Guessing Game

Guess a number between 1 and 100:

Sure, here are the instructions for the random number guessing game:

        1. **Objective**: The objective of the game is to guess the random number generated by the computer correctly.
        
        2. **Game Interface**:
           - When you open the game page, you'll see a heading "Random Number Guessing Game".
           - Below the heading, there will be a paragraph instructing you to guess a number between 1 and 100.
           - You'll also see an input field where you can enter your guess and a "Submit Guess" button to submit your guess.
           - After submitting a guess, the game will provide feedback on whether your guess is too high, too low, or correct.
           - A message will be displayed below the input field to inform you of the result of your guess.
        
        3. **How to Play**:
           - Enter your guess in the input field.
           - Click the "Submit Guess" button to submit your guess.
           - The game will tell you if your guess is too high, too low, or correct.
           - Keep guessing until you correctly guess the random number.
        
        4. **Winning**:
           - If you guess the correct number, the game will display a congratulatory message along with the number of guesses it took you to find the correct answer.
        
        5. **Restarting the Game**:
           - You can reload the page to start a new game and guess a new random number.
        
        6. **Have Fun!**:
           - Enjoy playing the game and see how quickly you can guess the correct number.
